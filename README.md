# LIFT Method Landing Page

A premium Next.js landing page built for high-net-worth retirement wealth planning.

## Features
- **High-End Design**: Glassmorphism, premium typography (Outfit/Inter), and luxury gold accents.
- **Copy-First Storytelling**: Following 'The LIFT Method' framework (Leverage, Indexed Growth, Flexible, Tax-Advantaged).
- **Responsive & Premium**: Optimized for conversion on any device.

## Assets
The project uses generated high-resolution assets for hero visuals, tax-trap analogies, and 'Double Duty Dollars' concepts.

## Getting Started
1. `npm install`
2. `npm run dev`

Deployment: Vercel recommended. Pushed to GitHub repo for seamless sync.
